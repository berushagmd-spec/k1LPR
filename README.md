# Kurbani Alert Bot v3

Исправления:

- команды теперь ловятся ручным роутером, а не только `CommandHandler`;
- добавлена команда `/id`;
- дефолтный интервал теперь 3–10 минут;
- после `/resume` и `/set_interval` следующий пост ставится почти сразу;
- `Мноройская область` исправлена на `Моноройская область`;
- `Диямская агломерация` вынесена отдельным субъектом;
- FPV сделаны очень редкими;
- БПЛА остаются редкими;
- баллистика стала отдельным и частым типом угрозы.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python bot.py
```

В `.env` вставьте токен BotFather. Бот должен быть админом канала и иметь право публиковать.

## Если команды не работают

Пишите команды в личку боту, не в канал.

Сначала отправьте:

```text
/id
```

Если ID отличается от `7787565361`, добавьте показанный ID в `ADMIN_IDS` в `.env` и перезапустите бота.

## Команды

`/status` — статус.  
`/preview` — пример без публикации.  
`/sample 5` — несколько примеров.  
`/post_now` — сразу пост в канал.  
`/custom текст` — свой пост.  
`/pause` — пауза.  
`/resume` — продолжить.  
`/set_interval 1 3` — интервал в минутах.  
`/mode calm|normal|hot|chaos` — интенсивность.  
`/scenario mixed|bpla|missile|ballistic|aviation|artillery` — сценарий.  
`/active` — активные ветки.  
`/clear_active` — очистить ветки.
