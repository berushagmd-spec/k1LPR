# Kurbani Alert Bot v6

Что добавлено:

- отдельный пост отбоя сирены:
  `✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅`;
- отдельная картинка для отбоя воздушной тревоги;
- если по опасности пришел текстовый отбой, следующий быстрый пост может дать именно отбой сирены;
- добавлена ручная команда для отправки отбоя сирены.

## Формат отбоя сирены

```text
Дияма
✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅
```

## Картинки

### Воздушная тревога
```env
AIR_ALERT_IMAGE_PATH=air_alert.jpg
AIR_ALERT_AS_PHOTO=true
```

### Отбой воздушной тревоги
```env
CLEAR_AIR_ALERT_IMAGE_PATH=clear_air_alert.jpg
CLEAR_AIR_ALERT_AS_PHOTO=true
```

Положите оба файла рядом с `bot.py`.

Если файлы найдены, бот будет отправлять:

- `⚠️ВОЗДУШНАЯ ТРЕВОГА⚠️` как фото с подписью;
- `✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅` как фото с подписью.

Если файл не найден, бот отправит обычный текст.

## Пример цепочки

```text
Фиксация ракет (ОТРК «Ураган-Т»)
В направлении: Диямская агломерация
```

Потом быстро:

```text
Дияма, Пуснарь и близлежащие
Диямская агломерация
БАЛЛИСТИЧЕСКАЯ УГРОЗА
Время реакции минимальное
```

Потом быстро:

```text
Дияма
⚠️ВОЗДУШНАЯ ТРЕВОГА⚠️
```

Позже:

```text
Дияма, Пуснарь и близлежащие
Диямская агломерация
ОТБОЙ БАЛЛИСТИЧЕСКОЙ УГРОЗЫ
```

Потом быстро:

```text
Дияма
✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅
```

## Команды

`/post_air Дияма` — вручную отправить воздушную тревогу.  
`/post_clear_air Дияма` — вручную отправить отбой воздушной тревоги.  
`/set_air_image air_alert.jpg` — путь к картинке тревоги.  
`/set_clear_air_image clear_air_alert.jpg` — путь к картинке отбоя.  
`/set_fast_interval 25` — быстрый интервал в секундах.  
`/clear_active` — очистить ветки, ожидающие фиксации и сирены.

После обновления лучше выполнить `/clear_active`.


## v7: падежи субъектов

Названия субъектов теперь всегда выводятся только в именительном падеже.

Правильно:

```text
Фиксация ракет (ракеты наземного базирования)
В направлении: Диямская агломерация
```

```text
Пролёт авиации (БР-24 «Горн»)
В направлении: Сугдинская область
```

Больше не будет вариантов вроде `Диямскую агломерацию`.


## v8: Ангерская область и БПЛА

`Ангераская область` заменена на `Ангерская область`.

БПЛА теперь генерируются только для ближайших зон:
- Форская область;
- Моноройская область;
- Алугданская область;
- Сугдинская область.

Для средних, дальних и глубоких зон БПЛА больше не выбираются. Там остаются авиация, ракеты и баллистика.


## v9: районы пуска, комбинированные атаки и ПВО

В постах о фиксации/пусках теперь указывается fictional-область Турбании, откуда идет пуск или вылет.

Пример:

```text
Фиксация ракет (крылатые ракеты)
Район пуска: удалено
В направлении: Диямская агломерация
```

Для авиации:

```text
Пролёт авиации (БР-24 «Горн»)
Район вылета: Турбанов-Ярская область
В направлении: Сугдинская область
```

Для комбинированной атаки:

```text
Фиксация комбинированной атаки (ракеты + авиация)
Ракетный компонент: КР «Корсар»
Авиационный компонент: БР-24 «Горн»
Район пуска/вылета: Новотурбанская область
В направлении: Диямская агломерация
```

Артиллерия по-прежнему не получает район источника.

Добавлен отдельный тип поста:

```text
Дияма
Работа ПВО
```

Он может появляться в срочной связке между опасностью и воздушной тревогой. Также есть ручная команда:

```text
/post_pvo Дияма
```

Добавлен сценарий:

```text
/scenario combined
```


## v10: правильные районы пуска Турбании

Список источников теперь жёстко ограничен.

Для обычных ракет и авиации:
- Новотурбанская область;
- Турбанов-ярская область.

Для баллистики:
- Лесопольский АО;
- Тихоборская Республика.

Старые неправильные источники удалены: Лахтинская область, Рекославская область, Дамирск-Саржинская область, Северо-Турбанская область.

Пример обычных ракет:

```text
Фиксация ракет (крылатые ракеты)
Район пуска: Новотурбанская область
В направлении: Диямская агломерация
```

Пример баллистики:

```text
Фиксация ракет (ОТРК «Ураган-Т»)
Район пуска: Лесопольский АО
В направлении: Сугдинская область
```

Пример комбинированной атаки:

```text
Фиксация комбинированной атаки (баллистика + авиация)
Ракетный компонент: ОТРК «Рубеж-М»
Авиационный компонент: И-30 «Коршун»
Район пуска: Тихоборская Республика
Район вылета: Турбанов-ярская область
В направлении: Ангерская область
```
