# Kurbani Alert Bot v6

Что добавлено:

- отдельный пост отбоя сирены:
  `✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅`;
- отдельная картинка для отбоя воздушной тревоги;
- если по опасности пришел текстовый отбой, следующий быстрый пост может дать именно отбой сирены;
- добавлена ручная команда для отправки отбоя сирены.

## Формат отбоя сирены

```text
Дияма
✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅
```

## Картинки

### Воздушная тревога
```env
AIR_ALERT_IMAGE_PATH=air_alert.jpg
AIR_ALERT_AS_PHOTO=true
```

### Отбой воздушной тревоги
```env
CLEAR_AIR_ALERT_IMAGE_PATH=clear_air_alert.jpg
CLEAR_AIR_ALERT_AS_PHOTO=true
```

Положите оба файла рядом с `bot.py`.

Если файлы найдены, бот будет отправлять:

- `⚠️ВОЗДУШНАЯ ТРЕВОГА⚠️` как фото с подписью;
- `✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅` как фото с подписью.

Если файл не найден, бот отправит обычный текст.

## Пример цепочки

```text
Фиксация ракет (ОТРК «Ураган-Т»)
В направлении Диямскую агломерацию
```

Потом быстро:

```text
Дияма, Пуснарь и близлежащие
Диямская агломерация
БАЛЛИСТИЧЕСКАЯ УГРОЗА
Время реакции минимальное
```

Потом быстро:

```text
Дияма
⚠️ВОЗДУШНАЯ ТРЕВОГА⚠️
```

Позже:

```text
Дияма, Пуснарь и близлежащие
Диямская агломерация
ОТБОЙ БАЛЛИСТИЧЕСКОЙ УГРОЗЫ
```

Потом быстро:

```text
Дияма
✅ОТБОЙ ВОЗДУШНОЙ ТРЕВОГИ✅
```

## Команды

`/post_air Дияма` — вручную отправить воздушную тревогу.  
`/post_clear_air Дияма` — вручную отправить отбой воздушной тревоги.  
`/set_air_image air_alert.jpg` — путь к картинке тревоги.  
`/set_clear_air_image clear_air_alert.jpg` — путь к картинке отбоя.  
`/set_fast_interval 25` — быстрый интервал в секундах.  
`/clear_active` — очистить ветки, ожидающие фиксации и сирены.

После обновления лучше выполнить `/clear_active`.
